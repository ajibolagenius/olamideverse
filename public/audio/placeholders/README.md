# Placeholder Audio Files

This directory contains placeholder audio files used throughout the OlamideVerse application for development and testing purposes.

## Audio File Types

### Track Audio Files
- `science-student.mp3` - Science Student track audio
- `wo.mp3` - WO!! track audio
- `lagos-nawa.mp3` - Lagos Nawa track audio
- `motigbana.mp3` - Motigbana track audio
- `bugle.mp3` - Bugle track audio

### Preview/Stream Files
- `wo-stream.mp3` - WO!! stream audio
- `science-student-stream.mp3` - Science Student stream audio
- `triumphant-stream.mp3` - Triumphant stream audio
- `loading-stream.mp3` - Loading stream audio
- `preview.mp3` - Generic preview audio

### Preview Files for Services
- `wo-preview.mp3` - WO!! preview for Spotify
- `science-student-preview.mp3` - Science Student preview for Spotify

## Usage

These placeholder audio files are referenced throughout the codebase using paths like:
- `/audio/placeholders/filename.mp3`

Replace these with actual audio files when available. Preview files are typically 30-second snippets used for demos.
